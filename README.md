# QuesHeroes

Fonctionnalités :

-Choix de l'équipe de 3 parmis 6 personnages
-Combattre tout les monstres de la base de données dans l'ordre 
-Menu de combat(attaquer ou regarder equipe)
-Interface avec barre de vie pour les monstres et l'équipe
-choix des attaque parmis 3 pour chaque personnages avec cooldowns et effets
-appliquer des buff(soin, stat) ou des status(brulure, poison, saignement)
-obtention d'item après avoir vaincu un monstrre choisis en fonction de sa rareté
-interface graphique pygame
